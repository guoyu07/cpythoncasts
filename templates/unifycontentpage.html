from jeapsns.views import hello
urlpatterns = patterns('',
        (r'^hello/$', hello),

Symbol   Matches
. (dot)     一个单个的数字或者字母
\d         单个的数字
[A-Z]   大写字母
[a-z]   小写字母
[A-Za-z]    所有字母
+   一个或者更多的 (e.g., \d+ 1个或者多个数字)
[^/]+   /之前可以是很多字符
?   0或者1个 (e.g., \d? matches 0或者1个数字)
*   0，1，更多 (e.g., \d* 0，1，更多的数字)

Templates 使用模板可以把django 的动态数据替换到一个成型的网页里面。

例如当用户输入自己的名字的时候，你可以把它显示到你要的位置。

在views.py里面添加模板

from django.template import Template, Context
def index_temp(request,input_name):
        t = Template('My name is {{ name }}.')
        c = Context({'name': input_name})
        return HttpResponse(t.render(c))

t是一个模板。 其中{{name}} 里面的是变量。 我用c里面的字典key 'name' 的值来替换 t.

函数index_temp 是一个带参数的函数。

在 urls.py 添加如下内容

from jeapsns.views import hello,index,current_time,index_temp
(r'^index_temp/([a-z]{1,10})$',index_temp),

（[a-z]{1,10}）, 圆括号代表参数，[a-z]规则是字母，{1,10}个数是1-10个。

访问：

http://127.0.0.1:8000/index_temp/jike

显示：

My name is jike.

增加一个新的参数：

urls.py,views.py

 (r'^index_temp/([a-z]{1,10})/(\d{1,6})$',index_temp),

--
def index_temp(request,input_name,color):
        t = Template('My name is<font color =#{{color}}> {{ name }}.</font>')
        c = Context({'name': input_name,'color':color})
        return HttpResponse(t.render(c))

浏览： http://127.0.0.1:8000/index_temp/jike/009900
