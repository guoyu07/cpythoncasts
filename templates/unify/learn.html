课程列表：

  • Java基础语言

    该Java课程是给那些只有一些或没有任何编码经验的初学者准备的，该课程以练习为目标导向，强调用户实际动手解决问题的能力。同时用户可以学习如何快速、轻松的创建大量的Java应用。

    你将会学到？

    • Java基础语法
    • 面向对象编程
    • 标准函数库
    • 网络编程

  • Android开发

    Android是一个基于Linux并应用于智能移动设备，如手机、平板电脑的开源操作系统。该课程涵盖基础Android编程，同时它也包括Android应用开发中使用的一些高级知识。 Android编程是以Java语言作为基础的，如果你掌握了Java编程那么学习Android将是一件非常愉快的事情。

    你将会学到？

    • 了解基础Android编程
    • 构建Android UI
    • Android高级概念

  • C基础语言

    C语言是能让你和计算机进行交互的，世界上最优秀的语言之一，学会它不仅可以能让你穿衣吃饭，更能展示你强大的思想和逻辑，C语言是编程语言的根据，从这里开始你的编程之旅吧！

    Skill Level: ACourse Author: code parkCourse Score: 0

    你将会学到？

    • C语言基础语法学习。
    • 分支控制和循环控制。
    • 数组和指针。
    • 函数和结构体。

  • Objective-C语言（持续更新中）

    Objective-C是开发iOS应用所使用的语言，它并不是一门复杂的计算机语言，如果你没有学过其他面向对象的编程语言可能会感觉Objective-C比较难学。但是你不需要担心，我们将会使用最简单的术语让你明白这些基础概念，在学习此课程前你需要先学习《C语言基础》这门课程。

    你将会学到？

    • 掌握面向对象编程
    • Objective-C基础语法
    • 操作对象和内存管理
